# 🏛️ المستشار AI - منصة استشارات قانونية متكاملة

## 📋 نظرة عامة
منصة قانونية شاملة ومتطورة توفر استشارات قانونية ذكية مع نظام أدوار متعدد، مولد قوالب، ونظام دفع إلكتروني.

---

## ✨ الميزات الرئيسية

### 🤖 **نظام الذكاء الاصطناعي**
- استشارات قانونية فورية
- 9 شخصيات قانونية متخصصة:
  - مستشار قانوني
  - محامي
  - قاضي
  - محامي شركات
  - محامي جنائي
  - محامي أسرة
  - أخصائي عقود
  - باحث قانوني
  - خبير استئناف

### 📄 **مولد القوالب والنماذج**
- 4 أنواع رئيسية من القوالب:
  - عقد عمل
  - عقد شراكة
  - صحيفة دعوى
  - مذكرة قانونية
- إنشاء وثائق مخصصة
- تحميل PDF و Word

### 💰 **نظام الدفع والرصيد**
- شحن الرصيد عبر التحويل البنكي
- نظام طلبات الدفع
- مراجعة واعتماد المدفوعات
- خصم رصيد لكل استشارة

### 👥 **نظام الأدوار**
- **المستخدم**: استشارات ونماذج
- **المدير**: إدارة شاملة للنظام

### 🎨 **واجهة المستخدم**
- تصميم حديث ومتجاوب
- الوضع المظلم
- دعم اللغتين العربية والإنجليزية
- تنقل سهل بين الصفحات

## 🚀 **كيفية التشغيل**

1. **تشغيل الخادم المحلي:**
   ```bash
   cd legal-ai-vanilla
   python3 -m http.server 8000
   ```

2. **فتح المتصفح:**
   ```
   http://localhost:8000
   ```

3. **تسجيل الدخول:**
   - كمدير: `owner@law.com` / `owner`
   - كمستخدم: أي بريد إلكتروني آخر

## 💡 **طريقة الاستخدام**

### للمستخدمين:
1. تسجيل الدخول
2. شحن الرصيد من صفحة الدفع
3. طرح الأسئلة القانونية
4. إنشاء القوالب والنماذج

### للمدير:
1. تسجيل الدخول كمدير
2. مراجعة طلبات الدفع
3. قبول أو رفض الطلبات
4. مراقبة الإحصائيات
